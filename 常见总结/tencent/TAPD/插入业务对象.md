## 不熟练的点

  css实现table表格斑马纹, 本质上是利用了css的子过滤器:nth-child配合odd或even实现奇偶行的背景色变化

```css
tr:nth-child(even) {
    background: #f1f1f1;
}

tr:nth-child(odd) {
    background: #f8f8f8;
}
```
